# Final Exam Review

A repository containing the mdbook files for B2 and B3 discussion section final exam review.

[![View the mdBook](https://img.shields.io/badge/mdBook-b1--final--exam--review-blue)](https://cdsds210.github.io/b1-final-exam-review/)

To host the website locally, first download the mdbook rust crate:

```bash
cargo install mdbook
```

Then locally host the website using

```bash
git clone https://github.com/cdsds210/b1-final-exam-review
cd b1-final-exam-review
mdbook serve --open
```
