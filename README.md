# Final Exam Review

A repository containing the markdown files for B2 and B3 discussion section final exam review.

To run locally, first download the mdbook rust crate:

```bash
cargo install mdbook
```

Then locally host the website using

```bash
git clone https://github.com/cdsds210/b1-final-exam-review
cd b1-final-exam-review
mdbook serve --open
```
